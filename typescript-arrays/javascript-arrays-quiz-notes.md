# javascript-arrays-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What are arrays used for?
  Organizing data
- Describe array literal notation.
  array declaration equals open bracket, into data values separated by commas, followed by close bracket
- How are arrays different from "plain" objects?
  arrays are ordered, have built in methods to interact with its data, and can be easily iterated over.
- What number represents the first index of an array?
  0
- What is the `length` property of an array?
  how many data values are stored in the array
- How do you calculate the last index of an array?
  array.length - 1
